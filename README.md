# 📈 TrendBoard
**TrendBoard** is a real-time financial news dashboard that delivers the latest market-moving stories with a clean, newspaper-style UI.
---
## 🚀 Features
* 📰 Live financial news feed
* 🔎 Search headlines, summaries, sources
* 🏷️ Auto-categorized (Markets, Tech, Crypto, Economy, etc.)
* 📊 Market ticker display
* 🎯 Category filters
* 📱 Fully responsive
---
## 🛠️ Tech Stack
* Next.js (App Router)
* TypeScript
* Custom CSS
* Vercel (Deployment)
## 🌐 Deployment
Deployed on Vercel
